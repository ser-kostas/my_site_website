Web Scraping

By Konstantinos Seretis
Last Updated On August 01

In this project, I developed a Python algorithm designed to scrape historical NBA data from the website Basketball Reference. The algorithm extracts data from three key tables: team standings, player stats per season, and MVP votes, covering all seasons from 1991 to 2020. This project demonstrates the power of web scraping to collect and analyze sports data efficiently, providing valuable insights into NBA history.
